# TAE-3 Java Assessment

This repository contains Java programs developed for **TAE-3 (Technical Assessment Exercise - 3)** based on Object-Oriented Programming concepts.

## Concepts Covered
- Classes and Objects
- Constructors
- Inheritance
- Polymorphism
- Abstraction
- Interfaces

## Programs Included
1. Person Class
2. Book and Library Management
3. Car Rental System
4. Resort Management System
5. Employee Salary Calculation
6. Rectangle Area Program

## Technologies Used
- Java
- VS Code
- Git & GitHub

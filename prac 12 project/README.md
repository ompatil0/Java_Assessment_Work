# ⏱️ Simple Stopwatch (Java OOP Project)

## 📌 Description

This project is a simple stopwatch application developed using **Java** and **Object-Oriented Programming (OOP)** concepts.
It allows the user to start and stop the stopwatch and displays the elapsed time.

---

## 🎯 Aim

To develop a menu-driven stopwatch application in Java using OOP concepts that calculates and displays elapsed time based on user input.

---

## 🛠️ Features

* Start stopwatch
* Stop stopwatch
* Display elapsed time in seconds
* Menu-driven interface
* Input validation (prevents invalid operations)

---

## 🧠 Concepts Used

* Class and Object
* Encapsulation
* Methods
* Conditional Statements
* Loop (do-while)
* Switch-case
* System Time (`System.currentTimeMillis()`)

---

```
Main.java
```

## ▶️ How to Run

### Step 1: Compile the program

```
javac Main.java
```

### Step 2: Run the program

```
java Main
```
